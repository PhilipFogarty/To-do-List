# To-Do List — Docker Compose

Aplicação de lista de tarefas com persistência em banco de dados, containerizada com Docker Compose.

---

---

## Funcionalidades

- Adicionar tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Persistência no banco de dados (as tarefas continuam após reiniciar)

---

## Tecnologias

- HTML5, CSS3, JavaScript
- Node.js + Express
- PostgreSQL
- Docker + Docker Compose
- Nginx

---

## Como rodar

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/seu-repo.git
cd seu-repo

# Suba os 3 serviços
docker compose up --build
```

Acesse **http://localhost** no browser.
